# EmployeManagementSystem

Full Stack Application using SpringBoot as Backend and React Front End using Mysql Database.

## Overview
This project is a comprehensive Employee Management System designed to streamline organizational workflows. It leverages a modern tech stack to provide a robust backend and a responsive frontend experience.

## Technical Stack
- Backend: Spring Boot
- Frontend: React.js
- Database: MySQL
- Communication: RESTful APIs

## Features
- Full CRUD (Create, Read, Update, Delete) functionality for employee records
- Seamless integration between React frontend and Spring Boot backend
- Persistent data storage using MySQL
- Scalable architecture suitable for organizational data management

## Getting Started
To get a local copy up and running, follow these steps:

### Prerequisites
- Java JDK 8 or higher
- Node.js and npm
- MySQL Server

### Installation
1. Configure the MySQL database connection in "src/main/resources/application.properties"
2. Build and run the Spring Boot application
3. Navigate to the frontend directory, install dependencies using "npm install", and start the development server using "npm start"
